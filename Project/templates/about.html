About AI Email Client

Welcome to the AI Email Client project! This project is designed to provide a feature-rich email client with AI-powered summarization capabilities.

Features

  • User Registration and Login
  • Profile Management
  • Compose, Send, and Receive Emails
  • Inbox and Sent Emails Management
  • AI-Powered Email Summarization using the Transformer Library
  • Password Reset Functionality

Technologies Used

  • Flask - Web Framework
  • Flask-SQLAlchemy - Database ORM
  • Flask-Bcrypt - Password Hashing
  • Flask-Login - User Session Management
  • Flask-Mail - Email Sending
  • Flask-Migrate - Database Migrations
  • Transformers - AI Summarization
  • Bootstrap - Frontend Framework

How to Use

To get started, register for an account or log in if you already have one. Once logged in, you can compose new emails, view your inbox, and manage your sent emails. Use the "Summarize" button to get a quick summary of any email.

Note: Passwords must be at least 8 characters long and contain a mix of letters and numbers.
